# AI Use Statement

## Purpose of This Statement

This document describes where and how AI tools were used in this project. It also clarifies what work was reviewed, modified, and validated by the project team.

## AI Tools Used

- Tool name: Github Co-Pilot, Claude Sonnet 4.6, Manus

---

## AI Usage

- Github Co-Pilot was used to generate bare bones layout for each frontend page.
- Github Co-Pilot was used to debug and perfect frontend page logic. This includes: timeslot booking logic, notification creation and interactive dashboards.  
- Manus was used to generate the styling for these pages, introducing new ui components and resources wherever needed.
- Claude was used to generate the Bruno test suite.
- Github Co-Pilot was used to generate repetitive endpoints to make developing repeat code easier.
- Github Co-Pilot and Claude we're used for 90% of debugging issues regarding microservices conenctions, networking issues, faults in logic, etc.
